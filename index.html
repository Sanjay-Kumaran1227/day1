Chapter 1: Introduction to Web Development

Web development is the process of building websites and web applications. It involves two major areas: Frontend (what users see) and Backend (server-side logic).

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB

The MERN stack combines these technologies to build full-stack applications efficiently.

Chapter 2: Understanding HTML

HTML stands for HyperText Markup Language. It is the backbone of every web page and defines the page structure.

Key HTML elements:

  1. <h1> to <h6> – headings
  2. <p> – paragraph
  3. <a> – links
  4. <img> – images
  5. <div> – containers

HTML describes *what* content is, while CSS decides *how* it looks.

Chapter 3: Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style HTML elements.

CSS allows you to control:

  • Colors
  • Fonts
  • Layouts
  • Spacing
  • Animations

Example CSS: 


h1 {
    color: blue;
    text-align: center;
}
